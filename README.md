# Cheat sheet terminal
A cheatsheet of commands for my workflow.

### Local
- cd
- cd ~/Desktop
- ls
- git status
### Working with a repo
- git clone [link]
- cd [folder]
- git add .
- git commit -m "commit"
- git push -u origin [branch]
### New repo
- git init
- git add --all
- git commit -m "commit"
- git push -u origin [branch]
### Branch and Merge
- git branch -av
- git branch [branch]
- git checkout [branch]
- git merge [branch]
